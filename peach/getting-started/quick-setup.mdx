## Step 1: Create your account

Sign up at [acme.com/signup](https://acme.com/signup) and verify your email.

## Step 2: Install the CLI

```bash
npm install -g @acme/cli
```

## Step 3: Initialize your project

```bash
acme init my-project
```

You're all set! Head to [First Project](/getting-started/first-project) to start building.
